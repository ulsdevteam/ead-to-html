# EAD to HTML XSLT

This stylesheet converts ArchivesSpace EAD (version 2) to HTML (for use in Islandora).

## Requirements

* XSLT 1.0
* PHP extensions

## Usage

Key features include wrapping nested containers in fieldsets to allow expand/collpase; actuating DAO xlinks; implementing OpenURL links to Aeon.
